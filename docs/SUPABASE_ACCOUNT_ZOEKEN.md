## ‚úÖ Zodra Je Het Project Hebt Gevonden

### Stap 1: Check Project Status
1. Ga naar het project dashboard
2. Check of het project gepauzeerd is
3. Als gepauzeerd: klik op **"Restore project"**

### Stap 2: Voer Migraties Uit
1. Ga naar **SQL Editor**
2. Voer migraties uit (zie `scripts/KOPIEER_PLAK_MIGRATIES.md`)

### Stap 3: Test Connectie
1. Voeg credentials toe aan `.env.local`
2. Herstart development server
3. Test Admin Dashboard

---

## üîê Als Je Niet Kunt Inloggen

### Optie 1: Password Reset
1. Ga naar: https://supabase.com/dashboard
2. Klik op **Forgot Password**
3. Voer email adres in
4. Check email voor reset link

### Optie 2: Nieuw Project Aanmaken
Als je het project niet meer kunt vinden:
1. Maak een nieuw Supabase project aan
2. Update `.env.local` met nieuwe credentials
3. Voer migraties uit in het nieuwe project
