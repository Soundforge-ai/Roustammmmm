- **Simpel gezegd:** Een tool van Google die laat zien hoe je site gevonden wordt
- **Wat het doet:** Toont welke woorden mensen gebruiken om je te vinden
- **Waarom belangrijk:** Je ziet waar je staat en wat je moet verbeteren

### Wat zijn rankings?
- **Simpel gezegd:** Op welke plek je staat in Google

- **Doel:** Zo hoog mogelijk (1-3 is het beste)
