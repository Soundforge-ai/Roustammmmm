### Stap 1: Open het Dashboard
1. Ga naar: `https://www.yannova.be/seo/rankings`
2. Je ziet een tabel met alle belangrijke keywords

### Stap 2: Test een Keyword (Handmatig)
1. **Open een incognito venster** (belangrijk voor niet-gepersonaliseerde resultaten)
   - Chrome/Edge: `Ctrl+Shift+N` (Windows) of `Cmd+Shift+N` (Mac)
   - Firefox: `Ctrl+Shift+P` (Windows) of `Cmd+Shift+P` (Mac)

2. **Ga naar Google.com**

3. **Zoek op het keyword**, bijv.: `ramen en deuren Keerbergen`

4. **Zoek je website** in de resultaten:
   - Scroll door de resultaten
   - Noteer op welke positie je staat (1, 2, 3, etc.)
   - Als je niet gevonden wordt, noteer "Niet gevonden"

### Stap 3: Voer de Positie In
1. **Terug in het dashboard**, zoek het keyword in de tabel
2. **Klik in het "Positie" veld** naast het keyword
3. **Voer het nummer in** (1-100, of laat leeg voor "Niet gevonden")
4. **De positie wordt automatisch opgeslagen** in je browser

## üí° Tips & Best Practices

### 1. Gebruik Altijd Incognito
- **Waarom?** Google personaliseert resultaten op basis van je geschiedenis
- **Hoe?** Open altijd een incognito venster voor accurate resultaten

### 2. Wees Consistent
- **Test op hetzelfde tijdstip** (bijv. elke maandag ochtend)
- **Gebruik dezelfde locatie** (of gebruik VPN voor specifieke locaties)

### 3. Noteer Details
- **Screenshot** belangrijke rankings
- **Noteer** welke concurrenten boven je staan
- **Check** of je featured snippets krijgt

### 4. Combineer met Google Search Console
- **Dashboard**: Handmatige tracking voor specifieke keywords
- **Search Console**: Automatische data voor alle keywords
- **Beide samen**: Volledig beeld van je SEO

### 5. Export Regelmatig
- **Maandelijks CSV export** voor backup
- **Bewaar historische data** voor trends
- **Gebruik voor rapportage** naar management

## üÜò Problemen Oplossen

### "Ik kan mijn site niet vinden"
1. **Check of site ge√Øndexeerd is:**
   - Zoek: `site:www.yannova.be`
   - Als niets verschijnt: site niet ge√Øndexeerd

2. **Oplossing:**
   - Gebruik Google Search Console URL-inspectie
   - Submit sitemap
   - Wacht 24-48 uur

### "Posities zijn inconsistent"
- **Oorzaak:** Google personaliseert resultaten
- **Oplossing:** Gebruik altijd incognito venster
- **Tip:** Test op verschillende momenten

### "Data verdwijnt"
- **Oorzaak:** Browser cache/localStorage gewist
- **Oplossing:** Export regelmatig naar CSV als backup
- **Tip:** Data wordt opgeslagen in browser, niet op server
